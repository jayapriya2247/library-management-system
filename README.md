# 📚 Library Management System

A full-stack **Library Management System** developed using **Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript**. The application enables librarians to efficiently manage books, members, and book issue/return operations through a simple and user-friendly interface.

---

## 🚀 Features

- 🔐 User Registration and Login
- 🔑 JWT Authentication
- 📖 Book Management (Add, Update, Delete, View)
- 👥 Member Management
- 📚 Issue and Return Books
- 📊 Dashboard Statistics
- 💾 MongoDB Database Integration

---

## 🛠️ Technologies Used

### Frontend

### Backend
- Node.js
- Express.js

## ▶️ How to Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/jayapriya2247/library-management-system.git
```

### 2️⃣ Navigate to the Backend Folder

```bash
cd library-management-system/backend
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Configure Environment Variables

Create a `.env` file inside the `backend` folder and add:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```

### 5️⃣ Start the Backend Server

```bash
npm start
```

### 6️⃣ Run the Frontend

Open the `frontend/index.html` file using **Live Server** in Visual Studio Code.
